# vue3-form

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles lib for production
```
yarn build:core
```

### Compiles form theme for production
```
yarn build:theme
```

### Run your unit tests
```
yarn test
```

### Lints and fixes files
```
yarn lint
```
